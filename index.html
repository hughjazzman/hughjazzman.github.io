CLIrcuit Assistant

CLIrcuit Assistant is a desktop app to implement and solve simple circuits, optimized for use via the Command Line Interface (CLI). The application was built with Java, applying Software Engineering techniques.

Knewbie

Knewbie was done as a summer project under Orbital, hosted by National University of Singapore (NUS) and supported by the likes of Google and PayPal. The platform was built having PSLE Mathematics students in mind, employing machine learning to provide tailored educational content to students for optimal learning. It is built using Python and the Flask framework.

From a total of 383 projects, Knewbie was 1 of 6 projects selected by the School of Computing (SoC) communication office, NUS and was featured on the various social media accounts.

RTOS-based Robotic Car

The aim of this project is to design a Real-Time Operating System (RTOS) based robotic car that is controlled through an Android App. The robotic car is able to establish a Bluetooth connection with the Android App, receive commands from the Android App and execute the correct response, move the car in multiple directions, control the various LED’s according to the car’s status, and play different sounds/tunes according to the cars status.

Since the above tasks must be carried out simultaneously, the implementation of the RTOS Keil RTX was required.

Alex

Alex is a Search and Rescue (SAR) robot built using Raspberry Pi, Arduino, Lidar, with a TLS client-server setup, using C/C++. The goal is to navigate an unseen room solely using commands sent via a laptop, mapping out the room environment via information sent from the robot.

Jane Street Trading Bot

This automated trading bot for a virtual exchange was built using Python by analysing and estimating fair value. Our team, NullPointerException, won the Last Hour Challenge.

FPGA Sound Display and Entertainment System

Using Verilog, we built a Sound & Entertainment System on a Basys3 board with an OLED display and a microphone attached. Together with my teammate, we programmed Pong, Tetris, and responsiveness to audio for an entertaining and interactive experience.
